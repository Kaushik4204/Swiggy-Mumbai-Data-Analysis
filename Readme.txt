# Swiggy Mumbai Restaurant Data Analysis

This project performs data analysis and visualization on restaurant listings in Mumbai, similar to those found on food delivery platforms like Swiggy. The dataset includes information about ratings, delivery times, cuisine types, cost, and discount offers.

---

#Project Objectives

- Analyze restaurant ratings, delivery times, cuisines, and discount patterns
- Clean and transform raw CSV data into structured SQLite format
- Generate business insights using SQL and Python visualizations

---

## Tech Stack

| Tool        | Purpose                          |
|-------------|----------------------------------|
| Python      | Core programming language        |
| Pandas      | Data manipulation and cleaning   |
| SQLite3     | Lightweight relational database  |
| Matplotlib  | Data visualization               |
| Seaborn     | Statistical plots                |
| Plotly      | Interactive charts               |
| VS Code     | IDE and SQLite Explorer plugin   |
